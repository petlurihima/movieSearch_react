# MovieSearch App

MovieSearch is a web application built using React and Vite that allows users to search for movies and display detailed information about them. The app fetches data from the OMDB API and displays movie information such as title, release year, actors, plot, rating, and country.

## Features

- Display a default set of movies on initial load.
- Search for movies by title.
- Display detailed information about each movie.
- Responsive layout with movies displayed in a grid.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher) or yarn (v1.22 or higher)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/petlurihima/movieSearch_react
   cd moviesearch
2.Install dependencies:
   npm install
3.Run the development server:
   npm run dev
4.Open your browser and navigate to http://localhost:3000 to see the app in action.
